# Mandarin Vocabularies

A premium Next.js study tool that beautifully displays Mandarin Chinese vocabulary from the HSK 2.0 standards (Levels 1 to 6).

## Features
- **Dynamic Routing:** Blazing fast dynamic generation of HSK 1-6 vocabulary.
- **Micro-Animations:** Fluid micro-animations across vocabulary cards and selectors.
- **Glassmorphism Design:** Modern UI principles ensuring comfortable focus.
- **Text-to-Speech Audio:** Hear character pronunciation with a single click natively within the browser using Web Speech API.

## Built With
- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- Vanilla CSS Custom Variables

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) in your browser to start studying!
